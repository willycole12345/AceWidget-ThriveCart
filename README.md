# AceWidget-ThriveCart

# Requirements
- PHP >= 8.1
- Composer
- PHPUnit (optional for testing)

### How to Run
1. Clone the repository ( https://github.com/willycole12345/AceWidget-ThriveCart.git )

2. Install dependencies
   composer install

3. Run the test script
   php index.php

4. Run PHPUnit tests
   ./vendor/bin/phpunit tests/test
